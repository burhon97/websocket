# WebSocket

This example chat with two client with `WebSocket`.

## Usage

1. Lets start our server:

```bash
cd ./server
npm run compile
npm start
```

2. Start client 1:

```bash
cd ./client-1
npm run compile
npm start
```

3. Start client 2:

```bash
cd ./client-2
npm run compile
npm start
```
